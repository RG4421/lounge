API Documentation

Document

Base Document class. Represents an instance of a model. Handles all generic model functionality.

CouchbaseDocument

Inherits drom Document and handles all the database related functionality of a Model.

Model

Model inherits from Couchbase Document and is the base of all Models created using Lounge. This also has the Model creation functions.
